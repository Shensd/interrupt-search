What do all the search fields mean?

Field Name Description
category In the original list, interrupts were each assigned a category (list below), or if they were given none I gave them the category of 'uncategorized.' This field searches categories case insensitive via contains, so full category names are not required.
number When calling an interrupt, a specific number is usually put in the AH/AX register to specify the specific desired function.
vector This is the number used when calling INT, numbers are encoded in hex.
tagline In the original list, each interrupt was given a sort of tagline, or a breif summary usually containing the category and system it was for. This field searchs that propery case insensitive via contains.
section content This field is for searching the section content of a given interrupt, or the extra information such as description, returns, registers, and anything else noted by Ralf.
